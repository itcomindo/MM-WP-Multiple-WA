MM Multiple WA merupakan plugin Wordpress untuk menghadirkan fitur multiple whatsapp seperti diantaranya:

- Menampilkan lebih dari 1 nomor whatsapp
- Pilihan menampilkan tombol whatsapp saja atau telepon saja atau email saja atau diantaranya bahkan semuanya
- Pilihan untuk upload logo perusahaan
- Pilihan untuk menampilkan nama perusahaan
- Pilihan untuk menampilkan nama perusahaan diatas atau dibawah tombol whatsapp
- Pilihan untuk upload photo customer service
- Pilihan warna background, warna text, warna tombol dll
- Pilihan untuk disable tombol pada hari-hari tertentu
- Pilihan custom text untuk tiap-tiap tombol
- Pilihan custom text untuk frontpage, category, tag, single post, page, produk dst
- Pilihan posisi whatsapp: bottom left, bottom right, top left, top right, center left, center right, center top, center bottom termasu floating
- Plihan menampilkan tombol menggunakan shortcode
